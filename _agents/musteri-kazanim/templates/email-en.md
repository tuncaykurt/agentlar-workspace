# 📧 English Email Templates — Customer Acquisition Agent

> **Location:** `_agents/musteri-kazanim/templates/email-en.md`
> **Usage:** Used when campaign config has `sablon_dili: "EN"`

---

## 📋 Variable Reference

| Variable | Description | Example |
|----------|------------|---------|
| `{first_name}` | Recipient's first name | Sarah |
| `{brand_name}` | Brand/company name | Acme Corp |
| `{platform}` | Source platform | Instagram |
| `{niche}` | Content niche | AI, tech |
| `{value_prop}` | Campaign value proposition | Collaboration opportunity |
| `{sender_name}` | Sender's name | Your Name |
| `{cta}` | Call to action | Reply to discuss details |
| `{hook}` | Personalized opening hook | Loved your latest review |
| `{social_proof}` | Metrics/results | X views, Y engagement |

---

## 1️⃣ Brand Outreach — Personal Contact

### Subject:
```
Collaboration idea for {brand_name}
```

### Body:
```
Hi {first_name},

I'm {sender_name}, a content creator focused on {niche}. {social_proof}

My profiles:
- Instagram: YOUR_INSTAGRAM_URL
- TikTok: YOUR_TIKTOK_URL
- YouTube: YOUR_YOUTUBE_URL

{hook}

{cta}

Best,
{sender_name}
```

---

## 💡 Personalization Rules

1. **Sound natural** — conversational, not corporate
2. **Opening hook max 2 sentences**
3. **Be specific** — reference actual content
4. **Social proof** — lead with metrics
5. **One CTA per email**
6. **A/B test** subject lines
