# 🧠 Knowledge Base

Bu klasör, Antigravity ile çalışırken birikmesi gereken kalıcı bilgileri içerir.
Her yeni konuşmada bu klasördeki dosyalar bana bağlam sağlar; sen tekrar tekrar aynı şeyleri açıklamak zorunda kalmazsın.

---

## Klasör İçeriği

| Dosya | İçerik |
|---|---|
| `api-anahtarlari.md` | Kullandığın tüm API anahtarları ve servisler |
| `calisma-kurallari.md` | Tercihleriniz, tekrarlayan talepleriniz, çalışma tarzınız |
| `hatalar-ve-cozumler.md` | Geçmişte karşılaşılan hatalar ve çözümleri |
| `deploy-registry.md` | Railway'e deploy edilen projelerin ID kayıtları |
| `maliyet-takibi.md` | API servisleri maliyet ve kota takibi |
| `credentials/` | 🔐 Merkezi şifre/token deposu (git'te yok) |

---

## Nasıl Kullanılır?

- **Yeni bir şey eklemek istediğinde:** Doğrudan bu dosyaları düzenle ya da benden düzenlememi iste.
- **Yanlış bir bilgi varsa:** "\_knowledge/api-anahtarlari.md dosyasındaki X bilgisini güncelle" de.
- **Yeni bir kural eklemek istersen:** `calisma-kurallari.md` dosyasına ekleyebiliriz.
- **Hata çözüldüğünde:** `hatalar-ve-cozumler.md` dosyasına otomatik kaydedilir.
