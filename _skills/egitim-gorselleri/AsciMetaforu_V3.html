Antigravity Kavramları — Gerçek Hayattan

Aşçı & Editör

AI sistemini anlamak için gerçek hayattan 2 karakter çağırdık. Agent, Skill, Knowledge, Workflow ve Project kavramlarını hiç teknik terim kullanmadan — fotoğraflarla keşfedelim.

1

Agent — Kişinin Kendisi

İşi yapan kişidir. Beynini, becerilerini ve bilgisini kullanarak görevleri tamamlar.

👨‍🍳 AGENT

Aşçı

Mutfakta görevli kişidir. Tarife bakar, malzemeleri hazırlar ve lezzetli yemekler üretir. Tüm mutfak sürecinin odak noktası.
🎬 AGENT

Editör

Video prodüksiyonunun yaratıcı gücüdür. Ham görüntüleri alır, işler ve izlenmeye hazır içerikler üretir.

3

Knowledge — İşletmeyi Tanıma Bilgisi

Agent'ın çalıştığı işletmeyi, kuralları ve erişim bilgilerini (API anahtarları) bilmesidir.

📖 KNOWLEDGE

👨‍🍳 Aşçı Neyi Bilir?

📋 Menü & Kurallar

Restoranın menüsünü, müşteri tercihleri, alerji listesini ve tedarikçi bilgilerini bilir.

🔑 Erişim Şifreleri

Hangi malzeme deposunun şifresi ne — hepsi Knowledge'dadır.

🔑 Depo erişim şifresi = API Key
📖 KNOWLEDGE

🎬 Editör Neyi Bilir?

🎨 Kanal Stili

Kanalın stilini, intro/outro'yu, brand renklerini ve tipografiyi bilir.

🔑 Arşiv Erişimi

Video arşivine, stock footage kitaplığına ve müzik lisans hesabına nasıl erişeceğini bilir.

🔑 Video arşiv erişimi = API Key

🔐 API Anahtarları Nedir?

Knowledge, Agent'ları sadece birer "becerili kişi" olmaktan çıkarıp sizin işletmenizin çalışanı haline getiren bilgidir. Gerçek dünyada kapı anahtarı ne ise, dijital dünyada API anahtarı odur — editörün videolara, aşçının tariflere erişmesini sağlayan dijital şifrelerdir.

Project — Tüm Bunları Kapsayan Yapı

🚀 PROJECT

📖 Knowledge: Menü & Kurallar
