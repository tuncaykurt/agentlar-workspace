Knowledge — İşletmeyi Tanıma Bilgisi

Agent'ın çalıştığı işletmeyi, kuralları ve erişim bilgilerini (API anahtarları) bilmesidir.

📖 Knowledge Nedir?

Knowledge; Agent'ları sadece birer "becerili kişi" olmaktan çıkarıp sizin işletmenizin çalışanı haline getiren bilgidir. Tıpkı işe yeni başlayan birine "şirket el kitapçığı" vermek gibi — ama içinde API anahtarları, şifreleri ve erişim bilgileri de var.

👨‍🍳Aşçı Neyi Bilir?

Restoranın menüsünü, müşteri tercihleri, alerji listesini ve tedarikçi bilgilerini bilir. Hangi malzeme deposunun şifresi ne — hepsi Knowledge'dadır.

🔑 Depo erişim şifresi = API Key
🎬Editör Neyi Bilir?

Kanalın stilini, hangi intro/outro'nun kullanılacağını, brand renklerini ve video arşivine nasıl erişeceğini bilir.

🔑 Video arşiv erişimi = API Key
📋Prodüktör Neyi Bilir?

Şirketin bütçe limitlerini, takvimini, ekip üyelerinin iletişim bilgilerini ve proje yönetim araçlarının giriş şifrelerini bilir.

🔑 Proje yönetim aracı şifresi = API Key
🔐API Anahtarları

Gerçek dünyada kapı anahtarı ne ise, dijital dünyada API anahtarı odur. Editörün videolara, aşçının tariflere, prodüktörün proje araçlarına erişmesini sağlayan dijital şifrelerdir.

🔑 Her erişim = Bir API Key
